# Vortex - Intelligent Advisor for Car Showroom Executives

## Project Structure

- Frontend: React + Vite app in the `Frontend` folder
- Backend: FastAPI service in the `Backend` folder

## Quick Start

### Frontend

1. Go to the `Frontend` folder
2. Install dependencies and run the dev server

### Backend

1. Go to the `Backend` folder
2. Run the FastAPI app (e.g. `uvicorn Backend.main:app` from the repo root, or `uvicorn main:app` from inside `Backend`)
